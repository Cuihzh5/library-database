# 📚 高校图书馆管理系统

## 项目简介
一个完整的MySQL数据库应用系统，用于管理高校图书馆的日常运营，包括图书借阅、预约、归还、续借、罚款计算、座位预约等核心功能。

## ✨ 功能特性
- ✅ **读者管理**：注册、信息维护、权限控制
- ✅ **图书管理**：CRUD操作、库存管理、位置管理
- ✅ **借阅管理**：借书、还书、续借、逾期处理
- ✅ **预约系统**：图书预约、座位预约
- ✅ **罚款系统**：自动计算、支付管理
- ✅ **统计分析**：借阅统计、热门分析、读者行为分析
- ✅ **报表生成**：多种统计报表输出

## 🛠️ 技术栈
- **数据库**: MySQL 8.0+
- **编程语言**: SQL
- **工具**: MySQL Workbench, draw.io
- **版本控制**: Git/GitHub
